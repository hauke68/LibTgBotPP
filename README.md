# LibTgBotPP
A simple Telegram Bot C++ library

# Requirements

To compile, you just need a C++-Compiler and CMake and some libraries:
* libstdc++-dev
* libcurlpp-dev
* libjsoncpp-dev

and - of course - their dependencies.

To compile and install the lib:
* mkdir build
* cd build/
* cmake ../
* make
* sudo make install
* sudo ldconfig

To use the library, you need:
* libcurlpp
* libjsoncpp
